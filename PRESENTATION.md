## Topics

1. Overview of the project (A)
    - 1.1 Simulated bank data for individuals and companies, including transactions and stock holdings
    - 1.2 Admin dashboard for browsing and searching individual and company data, big picture view of all the data
    - 1.3 Individuals can log in and view their own data and data for their company
    - 1.4 Login, roles, and dashboard access (JWT cookies + routing by role)
2. Tech stack (A)
    - 2.1 Frontend: HTML, CSS, JavaScript, HTMX, Chart.js
    - 2.2 Backend: Python, SQLAlchemy, FastAPI, Jinja2, Pydantic, PyJWT
    - 2.3 Database: MariaDB on Docker
    - 2.4 Database design (M)
    - 2.5 Authentication and authorization
3. Features/demo (M)
    - 3.1 Admin dashboard
    - 3.2 Company dashboard
    - 3.3 Individual dashboard
    - 3.4 Show log (A)
4. AI chatbot features (A)
    - 4.1 Prompt assembly
    - 4.2 JSON response
    - 4.3 Python database querying
    - 4.4 Dashboard charts generation with httpx
