2. Tech stack (A)
    - 2.1 Frontend: HTML, CSS, JavaScript, HTMX, Chart.js
    - 2.2 Backend: Python, SQLAlchemy, FastAPI, Jinja2, Pydantic, PyJWT
    - 2.3 Database: MariaDB on Docker
    - 2.4 Database design (M)
    - 2.5 Admin dashboard: details (ORM, charting, HTMX swaps)
    - 2.6 Logging
3. AI chatbot features (A)
    - 3.1 Prompt assembly
    - 3.2 JSON response
    - 3.3 Python database querying
