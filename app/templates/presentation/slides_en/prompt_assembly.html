Prompt assembly

How does it work?

The prompt is built dynamically with context:

  • User context: Role and user ID
  • Page context: Current page for relevant visualizations
  • Database schema: Full database structure
  • Available tools: List of visualization keywords
  • Response schema: Exact JSON format expected

Prompt ingredients

The prompt includes the following information:

  • APP_HEADER: Core role of the AI assistant
  • Current page: Current page context
  • Current user: Role, person_id, company_id, username
  • Database schema: Full database structure
  • Allowed visualizations: List of available keywords
  • Chart rules: Rules for chart types and fields
  • Response contract: Exact JSON format schema
