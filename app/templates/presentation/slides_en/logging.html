Logging

File logging

Logs are written to daily rotated files:

logs/
  |-- 2025_11_19.log
  |-- 2025_11_20.log
  \-- 2025_11_25.log
  • Daily rotation: New file per day
  • Structured format: Timestamp, level, module, message
  • Terminal viewing: tail -f logs/*.log

Context-aware logging

Logs include contextual metadata:

from app.core.logger import log_context

log_context.bind(user_id=123, action="load")
LOGGER.info("Loading data")
# Output: user_id=123 action=load Loading data
  • Metadata: user_id, request_id, etc. added automatically
  • Traceability: Find all logs for a single action
  • Debugging: Full context for investigation
