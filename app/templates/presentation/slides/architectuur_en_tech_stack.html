Architectuur & tech stack

Backend

  • FastAPI: Web framework met async/await
  • SQLAlchemy: ORM voor database queries
  • Jinja2: Server-side rendering
  • PyJWT: Authenticatie

Database

  • MariaDB: Relationele database in Docker (3NF end BCNF)

Frontend

  • HTML/CSS: Semantische HTML, responsive design
  • HTMX: Interactiviteit zonder complexe JavaScript
  • Chart.js: Grafieken en visualisaties

HTMX

We zouden eerst FIXI.js gebruiken, maar HTMX werkt op dezelfde manier. Het idee is dat we geen JavaScript nodig hebben om de interactie te implementeren. We kunnen alles doen met HTML attributen en HTMX zorgt ervoor dat we geen pagina refresh nodig hebben.

  • Dashboard panel switches: Wanneer een user op "Individuals", "Companies", etc. klikt, wordt alleen het panel vervangen zonder dat de hele pagina hoeft te reloaden
<button
  hx-get="/dashboard/panel/individuals"
  hx-target="#dashboard-panel"
  hx-swap="innerHTML"
>
  Individuals
</button>

Chart.js

Chart configuraties worden server-side gegenereerd en via data attributes doorgegeven:

<div data-chart data-chart-type="pie"
     data-chart-config='{"labels": [...], 
                        "values": [...]}'>
  <canvas></canvas>
</div>
  • Server-side config: Data via JSON in HTML data attributes
  • Automatische initialisatie: Charts worden automatisch gerenderd
  • Theme-aware: Charts passen kleuren aan bij theme switches
