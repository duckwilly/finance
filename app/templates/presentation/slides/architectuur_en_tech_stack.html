Architectuur & tech stack

Backend stack

Het platform is gebouwd op een moderne Python backend met een duidelijke scheiding van verantwoordelijkheden.

  • FastAPI: Moderne, snelle web framework op Python 3.11 met automatische API documentatie en async/await support.
  • Routers: Georganiseerd in aparte modules voor dashboards, authenticatie, individuen, bedrijven en presentaties.
  • SQLAlchemy 2.x: Moderne ORM voor database interacties met type hints ondersteuning.
  • Pydantic v2: Data validatie en serialisatie voor request/response modellen.
  • PyJWT: Authenticatie en autorisatie via JWT tokens voor sessiebeheer.
  • Jinja2: Template engine voor server-side rendering van alle dashboards.

Database en persistentie

De data wordt opgeslagen in een relationele database met ondersteuning voor migraties en seed data.

  • MariaDB: Relationele database in Docker voor betrouwbare data opslag en complexe queries.
  • Docker Compose: Gecontaineriseerde database voor eenvoudige setup en consistentie.
  • Alembic: Database migratie tool voor versie-gecontroleerde schema wijzigingen.
  • Session management: SQLAlchemy sessions per request via dependency injection met connection pooling.
  • Seed data: Synthetische data gegenereerd via scripts en geladen via CSV bestanden.

Architectuur patronen

Het platform volgt duidelijke architectuur patronen voor onderhoudbaarheid en schaalbaarheid.

  • Service layer: Business logica gescheiden van routing en data access. Services encapsuleren berekeningen en aggregatie.
  • Dependency injection: FastAPI's dependency system voor database sessions, services en authenticatie.
  • Security middleware: Authenticatie middleware controleert requests en beschermt admin-only routes met JWT validatie.
  • Modulaire structuur: Code georganiseerd in aparte directories voor routers, models, schemas, services en database.
  • Startup warmup: Admin dashboard metrics geprecomputeerd bij opstarten voor snelle eerste laadtijden.
