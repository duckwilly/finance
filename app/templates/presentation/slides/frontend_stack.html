Frontend stack

HTML en CSS

De basis van de frontend is semantische HTML met custom CSS voor styling en layout.

  • Semantische HTML: Gebruik van betekenisvolle HTML elementen voor toegankelijkheid en SEO. Alle dashboards zijn server-side gegenereerd met Jinja2 templates.
  • Custom CSS: Modulaire CSS structuur met aparte stylesheets voor base, admin, login en presentation. Gebruikt CSS custom properties voor theming.
  • Responsive design: Flexbox en grid layouts zorgen voor een werkende interface op verschillende schermformaten, van desktop tot tablet.
  • Dark mode: Volledige ondersteuning voor dark mode via CSS custom properties. Gebruikers kunnen tussen light en dark mode switchen.
  • Component-based styling: Herbruikbare componenten zoals cards, buttons, lists en forms met consistente styling.

HTMX voor interactiviteit

HTMX voegt moderne interactiviteit toe zonder complexe JavaScript frameworks. Hier is een voorbeeld van hoe dashboard panels worden geladen:

<button
  type="button"
  hx-get="/dashboard/panel/individuals"
  hx-target="#dashboard-panel"
  hx-swap="innerHTML"
>
  Individuals
</button>

  • hx-get: Haalt HTML op via GET request
  • hx-target: Waar de response moet worden geplaatst
  • hx-swap: Hoe de content wordt vervangen (innerHTML, outerHTML, etc.)

Chart.js voor visualisaties

Chart configuraties worden server-side gegenereerd en via data attributes doorgegeven:

<div data-chart data-chart-type="pie"
     data-chart-config='{"labels": [...], 
                        "values": [...]}'>
  <canvas></canvas>
</div>

JavaScript initialiseert automatisch de charts:

const chart = new Chart(canvas, {
  type: 'pie',
  data: { labels: config.labels, 
          datasets: [{ data: config.values }] }
});
  • Server-side config: Data wordt via JSON in HTML data attributes gezet
  • Automatische initialisatie: Charts worden automatisch gerenderd bij page load
  • Theme-aware: Charts passen kleuren aan bij theme switches

Vanilla JavaScript

Minimale custom JavaScript voor specifieke functionaliteit. Bijvoorbeeld list filtering:

searchInput.addEventListener('input', () => {
  const term = searchInput.value.toLowerCase();
  rows.forEach(row => {
    const matches = row.dataset.searchText
      .includes(term);
    row.hidden = !matches;
  });
});

Of theme switching met localStorage:

function toggleTheme() {
  const isDark = root.dataset.theme === 'dark';
  root.dataset.theme = isDark ? 'light' : 'dark';
  localStorage.setItem('theme', root.dataset.theme);
}
  • Event-driven: Luistert naar user input en window events
  • State management: Gebruikt localStorage voor persistentie
  • DOM manipulation: Directe updates zonder frameworks
