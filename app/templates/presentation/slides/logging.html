Logging

File logging

Logs worden opgeslagen in bestanden met daily rotation:

logs/
  ├── 2025_11_19.log
  ├── 2025_11_20.log
  └── 2025_11_25.log
  • Daily rotation: Nieuw bestand per dag
  • Structured format: Timestamp, level, module, message
  • Live monitoring: tail -f logs/*.log

Context-aware logging

Logs bevatten contextuele metadata:

from app.core.logger import log_context

log_context.bind(user_id=123, action="load")
LOGGER.info("Loading data")
# Output: user_id=123 action=load Loading data
  • Metadata: user_id, request_id, etc. automatisch toegevoegd
  • Traceerbaarheid: Alle logs van een actie makkelijk vinden
  • Debugging: Volledige context voor probleemoplossing
