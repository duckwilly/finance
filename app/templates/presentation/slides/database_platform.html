Database platform

Seed data

Synthetische gebruikersdata wordt als csv gegenereerd met een pythonscript, en echte beursdata wordt ingeladen van yahoo finance:

# Genereer seed data
python scripts/gen_seed_data.py

# Laad data in database
python scripts/load_csvs.py
data/seed/
  ├── parties.csv
  ├── accounts.csv
  ├── journal_entries.csv
  └── holdings.csv

MariaDB op Docker

De database draait in een Docker container:

services:
  db:
    image: mariadb:11
    container_name: finance-mariadb
    environment:
      MARIADB_DATABASE: finance
      MARIADB_USER: app
      MARIADB_PASSWORD: apppwd
    ports:
      - "3306:3306"
    volumes:
      - db_data:/var/lib/mysql

Schema

Database schema wordt gedefinieerd in SQL:

-- sql/schema.sql
CREATE TABLE parties (
    id INT PRIMARY KEY AUTO_INCREMENT,
    party_type VARCHAR(20) NOT NULL,
    name VARCHAR(255) NOT NULL
);

CREATE TABLE accounts (
    id INT PRIMARY KEY AUTO_INCREMENT,
    party_id INT,
    FOREIGN KEY (party_id) REFERENCES parties(id)
);

Database connectie

SQLAlchemy maakt verbinding met de database:

def create_engine_from_settings():
    settings = get_settings()
    return create_engine(
        f"mysql+pymysql://{settings.database.user}:"
        f"{settings.database.password}@"
        f"{settings.database.host}:"
        f"{settings.database.port}/"
        f"{settings.database.name}"
    )

Connection string wordt opgebouwd met behulp van een .env:

DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=finance
DB_USER=app
DB_PASSWORD=apppwd
