Architecture

Architecture & tech stack

Layered FastAPI app with SQLAlchemy persistence and server-rendered UX enhanced by HTMX interactions.

Stack

  • Backend: FastAPI on Python 3.11 with routers for dashboards and authentication.
  • Persistence: MariaDB via SQLAlchemy 2.x and Alembic-ready migrations.
  • Presentation: Jinja2 templates, static CSS/JS, HTMX for partial swaps and smooth transitions.

Patterns

  • Service layer encapsulates metrics and chart assembly for admin panels.
  • Security middleware keeps admin-only surfaces scoped, while template responses stay cache-friendly.
  • Static assets are mounted at /static so the presentation can live inside the product chrome.
