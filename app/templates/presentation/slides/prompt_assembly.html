Prompt assembly

Hoe werkt het?

De prompt wordt dynamisch opgebouwd met context:

  • User context: Rol en ID van de gebruiker
  • Page context: Huidige pagina voor relevante visualisaties
  • Database schema: Volledige database structuur
  • Beschikbare tools: Lijst van visualisatie keywords
  • Response schema: Exact JSON format dat verwacht wordt

Prompt onderdelen

De prompt bevat de volgende informatie:

  • APP_HEADER: Basisrol van de AI assistant
  • Current page: Huidige pagina context
  • Current user: Rol, person_id, company_id, username
  • Database schema: Volledige database structuur
  • Allowed visualizations: Lijst van beschikbare keywords
  • Chart rules: Regels voor chart types en velden
  • Response contract: Exact JSON format schema
